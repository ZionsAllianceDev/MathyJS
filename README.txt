# MathyJS
a simple math engine for javascript 

Input 2 number in place of A and B to perform the function
MathyJS.add(a,b)
MathyJS.subtract(a,b)
MathyJS.divide(a,b)
MathyJS.multiply(a,b)

In the random function input 1 number in place of A
It will then confirm a random integer within that range
MathyJS.random(a)

Input 1 integer in the place of A to find it sqaure root
MathyJS.root(a)

Input 1 interger in the place of A to square or cube it
MathyJS.sqaure(a)
MathyJS.cube(a)

Input 1 integer in the place of A to find the division remainder
MathyJS.modulus(a)

Input 1 integer in the places of A and B to find to highest/lowest number
MathyJS.HighestNumber(a,b)
MathyJS.LowestNumber(a,b)
Input 1 integer in place of A to find out if it is odd or even
MathyJS.isOE(a)
Input 1 integer in place of A to find all prime number in its range
MathyJS.primeRange()
